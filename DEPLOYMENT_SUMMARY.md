## 🌐 部署信息

### GitHub 仓库

要部署到网络，你需要：

1. **创建 GitHub 仓库**
   - 访问：https://github.com/new
   - 仓库名：`labor-law-database`
   - 选择：Public
   
2. **推送代码**
   ```bash
   cd D:\Pheebs\labor-law-website
   git remote add origin https://github.com/YOUR_USERNAME/labor-law-database.git
   git branch -M main
   git push -u origin main
   ```

### Vercel 部署

1. **登录 Vercel**
   - 访问：https://vercel.com
   - 使用 GitHub 账户登录

2. **创建项目**
   - 点击：Add New → Project
   - 搜索：labor-law-database
   - 点击：Import

3. **部署**
   - 使用默认设置
   - 点击：Deploy
   - 等待 2-3 分钟

4. **获得网址**
   - 类似：https://labor-law-database.vercel.app
